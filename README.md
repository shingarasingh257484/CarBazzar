# CarBazzar
Car Bazar — Full-Stack Car Marketplace built with Node.js, Express, MongoDB, and EJS. Users can browse cars, filter by brand &amp; type, view details, register/login, and book cars. Admin can add, edit, and delete car listings.




#  Car Bazar – Car Marketplace Web Application

Car Bazar is a full-stack web application where users can browse, search, and book cars.  
The system supports authentication, car details, booking requests, and admin management.

---

##  Features

###  User Features
- Browse all available cars
- Filter by brand, type, price, and fuel
- View detailed specs, features & images
- Register & Login
- Book a car online

###  Admin Features
- Add new cars
- Update car details
- Delete cars
- Manage inventory (in stock / out of stock)

---
